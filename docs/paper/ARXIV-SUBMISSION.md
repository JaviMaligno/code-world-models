## 2. The submission bundle (upload this to arXiv)

`docs/paper/arxiv-submission/` contains the files arXiv needs:
- `main.tex`
- `main.bbl`  ← pre-generated so arXiv does NOT need to run bibtex
- `references.bib`  (included for completeness; arXiv uses the .bbl)
- `figures/` — the four PDF figures referenced by `\includegraphics`; arXiv
  **will not compile without them**.

A ready tarball is at `docs/paper/arxiv-submission.tar.gz`. Upload either the
tarball or the files directly (keep the `figures/` subdirectory).

To regenerate the bundle after any edit:
```bash
cd docs/paper
python ../../scripts/make_paper_figures.py   # refresh figures/*.pdf if data changed
pdflatex -interaction=nonstopmode main.tex && bibtex main && \
  pdflatex -interaction=nonstopmode main.tex && pdflatex -interaction=nonstopmode main.tex
mkdir -p arxiv-submission/figures
cp main.tex references.bib main.bbl arxiv-submission/
cp figures/*.pdf arxiv-submission/figures/
tar --exclude='._*' -czf arxiv-submission.tar.gz -C arxiv-submission \
  main.tex references.bib main.bbl \
  figures/danger_law.pdf figures/headline_play.pdf \
  figures/play_cost_mechanism.pdf figures/reach_divergence.pdf
rm -f main.aux main.log main.out main.blg main.toc main.bbl
```
